# SCART Invader 

An open-hardware RGB SCART multiplexer with features:

- Auto-detection of a video signal on CVBS, CSYNC, YPbPr, or S-Video
- Support for CVBS -> CSYNC with LM1881 on board (switched in or out by a relay) 
- IR remote control 
- Fully isolated relay multiplexer design 
- 100% Through-hole design for tedious hand assembly 
- Ultra-compact design 
- Enclosure printable on most home 3D printers 
- 2-layer PCB design for cheap manufacture by your favourite PCB fabricator
